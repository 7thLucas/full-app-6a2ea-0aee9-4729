# Location Tracker — Product

## Brand
- **Name:** Location Tracker
- **Tagline:** Your world, shared perfectly.
- **Voice:** Confident, adventurous, spatial, premium. Never technical. Celebrates the thrill of place.
- **Tone:** Direct and evocative — think explorer's journal meets premium cartography tool.

## Target Users
- **Tourists:** Travellers discovering new places who need to save and share them instantly.
- **Locals:** City insiders who want to share exact hidden-gem spots with visitors or friends.

## Core Features (MVP)
1. **Interactive 3D Globe** — Rotatable, zoomable photorealistic globe using Three.js or a Globe.gl library. The globe is the hero UI — front and center on load.
2. **Location Pinning** — Tap or click on the globe to drop a GPS pin. User can add a title and short description to the pin.
3. **Universal Share Links** — Every pin gets a clean, unique URL that opens and renders correctly on any platform or device with no app installation required on the recipient's side.
4. **One-Tap Cross-Platform Sharing** — Share pinned locations via WhatsApp, Instagram, X (Twitter), SMS, email, iMessage and any platform accepting a URL, via the device's native Web Share API or direct deep-link fallbacks.

## Strategic Principles
- Zero-friction sharing is the core promise — a broken link is a product failure.
- Visual delight on the globe is the primary retention hook.
- Reliability of shared links is non-negotiable; universal rendering is not optional.
- No turn-by-turn navigation at MVP.
- Open sharing model — no private/restricted lists at MVP.

## Anti-References
- Avoid cluttered map-app UIs (Google Maps, Apple Maps toolbar overload).
- Avoid flat, static 2D map aesthetics — everything must feel spatial and three-dimensional.
